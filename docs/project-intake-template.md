# 개인 프로젝트 정리 입력 양식

프로젝트가 완성된 뒤 아래 양식을 다른 AI에 전달해 사실에 맞게 채운 후, 결과를 그대로 전달합니다. 이 문서는 카드 앞면, 기술 스택 뒷면, README 모달의 네 섹션을 한 번에 구성하기 위한 입력값입니다.

## 다른 AI에 함께 전달할 요청문

```text
아래 프로젝트 사실만 사용해 포트폴리오 입력 양식을 채워줘.

- 모르는 내용, 수치, 성과, 배포 상태는 추측하거나 만들어내지 마.
- 문장은 한국어로 간결하게 작성해.
- 기술 이름만 나열하지 말고, 왜 선택했고 어떤 문제를 해결했는지 써줘.
- 배포하지 않았다면 배포 완료라고 쓰지 말고 현재 상태를 명확히 표시해.
- 코드나 HTML은 작성하지 말고, 양식의 항목만 채워줘.
```
